== Mission

Implémenter le fallback Tesseract (OCR sans IA) dans codex-gradle,
en respectant le boundary : codex détient les engines OCR documentaires
(READ), codebase orchestre la fallback chain LLM via un adapter.

== Items livrés

* `codex.ocr.OcrEngine` — interface `process(OcrRequest): OcrResult`
* `codex.ocr.TesseractOcrEngine` — CLI tesseract wrapper via ProcessBuilder, best-effort (ne jette pas), mapping langues ISO→tesseract
* Côté codebase : `CodexOcrEngineAdapter` wrap `OcrEngine` → `VisionProvider`, fallback chain Gemini → Ollama → Tesseract
