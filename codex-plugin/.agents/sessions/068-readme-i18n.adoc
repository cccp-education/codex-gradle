== Objectif

Créer des README multilingues (10 langues) pour les deux audiences de codex-gradle :
- *Consumer guide* (`README.consommateurs/`) — orienté utilisateur du plugin
- *Plugin internals* (`README.plugin/`) — orienté développeur/contributeur

Langues cibles (10) : EN, FR, ES, PT, ZH, HI, BN, AR, UR, RU.

=== Support RTL (Arabe + Urdu)

Les fichiers `README.ar.md` et `README.ur.md` sont en scripts RTL
(arabe/ourdou). Le contenu est intégralement traduit (titres, descriptions,
tables, commandes commentées). Aucun markup `dir="rtl"` HTML explicite —
le rendu Markdown GitHub gère le sens automatiquement par script Unicode.

== Décisions clés

- *2 audiences distinctes* : consumer guide (public-facing, focus pipeline
  métier) vs plugin internals (contributor-facing, focus build/CI/dépendances)
- *Master EN* marqué `<!-- master source — other languages are translations of this file -->`
- *Translations* marquées `<!-- translated from README.md rev 0.0.1 -->`
- *Pas de fichier README à la racine* `codex-gradle/` — les deux README vivent
  dans des sous-dossiers (`README.consommateurs/`, `README.plugin/`)
- *Aucune modification de code* — documentation uniquement
- *Pas de commit* — règle AGENTS.adoc (permission explicite requise)
